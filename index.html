Why Pin?

Minimal by design

Keep your notes close and simple. No servers, no lock-in, just plain JSON files under ~/.config/pin/.

Fast CLI workflow

Add and list entries instantly with a few keystrokes. Perfect for quick capture and retrieval.

Multiple databases

Create and switch between databases (e.g. ideas, todos, research) with a single command.

Zero-friction install

Just a Bash script plus jq. Optional figlet for a fancy help banner.

Install

  1. Ensure dependencies are installed:
macOS (Homebrew)
brew install jq figlet
Debian/Ubuntu
sudo apt-get update
sudo apt-get install -y jq figlet
Oneâ€‘line install (replace REPO URL if needed)
curl -fsSL https://raw.githubusercontent.com/itssrv/Pin/main/pin.sh | sudo tee /usr/local/bin/pin >/dev/null && sudo chmod +x /usr/local/bin/pin && pin help
  1. Make the script executable and put it on your PATH:
chmod +x pin.sh
# System-wide (may require sudo):
sudo cp pin.sh /usr/local/bin/pin

# Or user-local:
mkdir -p "$HOME/bin"
cp pin.sh "$HOME/bin/pin"
chmod +x "$HOME/bin/pin"
# Add to your shell rc, if needed:
# export PATH="$HOME/bin:$PATH"
  1. Verify:
pin help
